# greenpedia
## greenpedia website adalah website yang bergerak demi kemanusiaan dalam melestarikan lingkungan dengan cara yang positive yang dinaungi perusahaan greenpedia

#petunjuk instalasi
> siapkan vscodenya
> pastikan internet sudah aktif
> masuk ke vscode
> git clone repo ini

# cara menjalankan project
> masuk ke folder greenpedia yang sudah di git clone
> select index.html
> pada index.html klik kanan lalu open with live server
> jangan lupa install juga extensions live servernya
